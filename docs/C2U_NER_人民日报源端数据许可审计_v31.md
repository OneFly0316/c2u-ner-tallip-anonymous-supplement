# C2U-NER 人民日报源端数据许可审计（v31）

## 结论摘要

本文源端应表述为“公开流通的 1998 年《人民日报》（People's Daily）中文 NER 标注资源”，而不是简单写成“完全可自由再分发的公开数据”。原因是：原始 1998 年《人民日报标注语料库》涉及人民日报社新闻信息中心授权，北京大学计算语言学研究所与富士通研究开发中心参与制作；历史公开方式包含限定范围、有偿公开和免费公开 1 个月样例。后续 GitHub、数据集索引和 ModelScope 等平台存在二次整理版本，但不同版本的许可、字段和切分不完全一致。

## 已核查来源

| 来源 | 可支撑事实 | 对本文写法的影响 |
|---|---|---|
| 富士通中国 2001 新闻稿 | 1998 年全年 2600 万汉字《人民日报标注语料库》由北大计算语言学研究所、富士通研究开发中心在人民日报社新闻信息中心许可下共同制作；1998 年上半年约 1300 万字版本先在大学、研究所等限定范围有偿公开，并免费公开 1 个月样例 | 支撑“原始语料有明确权利方和历史发布限制”，不能仅因网络流通就默认可再分发 |
| InsaneLife/ChineseNLPCorpus GitHub | 收录“人民日报数据集”，说明人名、地名、组织名三种实体类型，并链接 1998 版本目录 | 支撑“存在公开流通的 1998 NER 整理版本”，但 GitHub 页面本身不等于原始版权许可 |
| SelectDataset `china-people-daily-ner` | 描述该数据集基于 1998 年人民日报新闻文本，包含 train/dev/test，约 8MB，并标注 CC-BY-SA-NC 4.0；页面给出的原始来源为 `http://s3.bmio.net/kashgari/china-people-daily-ner-corpus.tar.gz` | 可作为二次整理版本许可线索；若本文实际采用该版本，应记录下载页、访问日期和许可证 |

## 建议正文口径

1. 使用“公开流通的 1998 年《人民日报》中文 NER 标注资源”。
2. 表3列明“本文实际采用版本、下载页、获取日期、许可证原文、是否允许派生发布”。
3. 若采用 SelectDataset 或其他明确 CC-BY-SA-NC 4.0 的整理版本，可将 C2U-NER 目标端派生数据按非商业、署名、相同方式共享的限制发布。
4. 若无法确认本文本地文件来自哪个明确许可版本，则不要发布源端文本和目标端全文，只发布脚本、哈希、统计摘要、预测文件和复现说明。
5. 参考文献中应加入原始语料公开新闻稿和本文实际采用的数据整理版本 URL。

## 仍需作者最终确认

- 本地源端文件是否来自 InsaneLife、SelectDataset、ModelScope 或其他渠道。
- 下载或复制日期。
- 原始许可文件截图或网页归档。
- 是否允许机器翻译派生文本公开发布。
- 是否允许源端文本随补充材料公开。
